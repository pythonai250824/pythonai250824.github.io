Python Program

pwd: str = input("what's the password?");

if pwd == "python1234":
    print("correct! you are in!");
else:
    print("wrong! out you go ...");

## End

# Bonus:

## Tip: look at this way of checking--

PASSWORD: str = "python1234"

    print("correct! you are in!");
else:
    print("wrong! out you go ...");

This program checks if the input password matches "python1234" and prints a corresponding message.
