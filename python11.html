Basic Tuple Operations

Python Program

                
# Basic Tuple Creation and Access
# -- create tuple with 1 item using ,
group_max: tuple[int] = (100,)
print(group_max[0])  # Output: 100

simple_tuple: tuple[int] = (1, 2, 3)
one_item_tuple: tuple[int] = (1,)

100
("apple", "banana", "cherry")

This section covers the basics of creating and accessing tuple elements in Python. Tuples are immutable, meaning once created, their elements cannot be modified.

Tuple with Mixed Data Types

Python Program

                
# Tuples can hold elements of different data types
mix: tuple[any] = (1, "Hello", [1, 2, 3], {"name": "danny"})
print(mix)  # Output: (1, "Hello", [1, 2, 3], {"name": "danny"})

# Modifying a mutable element within a tuple
mix[2][0] = 100
mix[2].append(200)
print(mix)  # Output: (1, "Hello", [100, 200, 3], {"name": "danny"})

Run Demo

                
(1, "Hello", [1, 2, 3], {"name": "danny"})
(1, "Hello", [100, 200, 3], {"name": "danny"})

This section demonstrates that tuples can contain elements of various data types. While tuples themselves are immutable, if they contain mutable elements (like lists), those elements can be modified.

Tuple Unpacking and Nested Tuples

Python Program

                
# Unpacking Tuples
number_tuple: tuple[int] = (1, 2, 3)
x, y, z = number_tuple
print(x, y, z)  # Output: 1 2 3

create_nested_tuple: tuple[tuple[any]] = (
    ("Alice", 21, "Physics"),
    ("Bob", 222, "Chemistry"),

)

This section shows how to unpack tuples into individual variables and demonstrates the use of nested tuples to represent more complex data structures.

Advanced Tuple Operations

Python Program

                
# Creating Tuples with Ranges and Comprehensions
create_tuple = tuple(range(21))  # Tuple with numbers from 0 to 20
print(create_tuple)

# Tuple Comprehension Example
tuple_comp = tuple(x**2 for x in range(1, 6))
print(tuple_comp)

# Tuples from Lists and Zipping Tuples
names: tuple[str] = ("Alice", "Bob", "Charlie")
scores: tuple[int] = (80, 90, 99, 100)
combined_tuple: tuple[tuple[any]] = tuple(zip(names, scores))
print(combined_tuple)

This section explores advanced tuple operations such as creating tuples with ranges and comprehensions, and using the `zip` function to combine multiple sequences into tuples.

Sorting and Manipulating Tuples

Python Program

                
# Sorting Tuples
grades = (("Alice", 92), ("Bob", 90), ("Charlie", 88), ("David", 85), ("Eve", 79))
sorted_grades = tuple(sorted(grades, key=lambda pair: pair[1]))
print(sorted_grades)

# Tuple Concatenation and Repetition
tuple_1: tuple[int] = (1, 2)
tuple_2: tuple[int] = (3, 4)
print(tuple_1 + tuple_2)  # Output: (1, 2, 3, 4)
print(tuple_1 * 3)  # Output: (1, 2, 1, 2, 1, 2)

Run Demo

                
(("David", 85), ("Charlie", 88), ("Bob", 90), ("Alice", 92), ("Eve", 79))
(1, 2, 3, 4)
(1, 2, 1, 2, 1, 2)

This section demonstrates how to sort tuples, concatenate multiple tuples, and repeat tuples. These operations are useful for organizing and manipulating tuple data in Python.

datetime and timedelta

Python Program

                
from datetime import datetime, timedelta

print('Israel', datetime.now())  # Current local time
print('UTC   ', datetime.utcnow())  # Current UTC time
print(datetime.now().strftime("%H:%M:%S.%f %Y-%m-%d"))  # Format: Hours, Minutes, Seconds, Milliseconds, Year-Month-Day
print(datetime.now().strftime("%H:%M:%S.%f %Y-%B-%d %A"))  # Format: Hours, Minutes, Seconds, Milliseconds, Year-FullMonthName-Day FullDayName
print((datetime.now() + timedelta(days=1)).strftime("%H:%M:%S.%f %Y-%B-%d %A"))  # Time 1 day from now
print((datetime.now() + timedelta(days=30)).strftime("%H:%M:%S.%f %Y-%B-%d %A"))  # Time 30 days from now

Israel 2024-11-01 17:30:31.123456

17:30:31.123456 2024-11-01
17:30:31.123456 2024-November-01 Wednesday
17:30:31.123456 2024-November-02 Thursday
17:30:31.123456 2024-December-01 Friday

This section covers handling date and time in Python using the `datetime` module. The code demonstrates how to get the current local and UTC time, format date and time strings, and calculate future dates.

Using the `zip` Function

Python Program

                
names = ['Alice', 'Bob', 'Charlie']
ages = [25, 30, 35]

# Combine the two lists into a list of tuples
combined = list(zip(names, ages))
print(combined)

Run Demo

                
[('Alice', 25), ('Bob', 30), ('Charlie', 35)]

The `zip` function in Python is used to combine two or more iterables (e.g., lists, tuples) into a single iterable of tuples. Each tuple contains elements from the iterables that share the same index.

Using the `enumerate` Function

Python Program

                
fruits = ['apple', 'banana', 'cherry']

# Enumerate over the list
for index, fruit in enumerate(fruits):
    print(f"Index {index}: {fruit}")

Run Demo

                
Index 0: apple
Index 1: banana
Index 2: cherry

The `enumerate` function adds a counter to an iterable, returning it as an enumerate object. This is useful when you need to have both the index and the value of each item in a loop.
